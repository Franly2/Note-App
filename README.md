📌 Project Description

MySQL Node.js App A simple web application based on Node.js and Express that connects to a MySQL database.

This project uses EJS for templating, layout management with express-ejs-layouts, and supports CRUD (Create, Read, Update, Delete) operations via forms.

Intended as a learning example/portfolio piece to demonstrate skills in:

    MySQL configuration and connection in Node.js (mysql/mysql2),

    Express project structure with EJS views and layouts,

    Utilizing body-parser, method-override, and nodemon middleware for development.

🎯 Key Features:

    Displaying a list of data from a MySQL table.

    Add/edit data forms using method override (PUT/DELETE).

    Consistent layout using layouts/main-layout.ejs.

    Easy setup: run npm install followed by node app.js (or npm run dev if configured).
